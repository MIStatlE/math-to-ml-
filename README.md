## 为什么要建立这个库
在学习过程中，我常常感受到：  
- **信息过载**：资料和教材琳琅满目，筛选价值内容需要耗费大量时间。  
- **体系零散**：不同作者的符号和叙述方式，侧重点不一致，容易增加理解成本。  
- **缺乏路径感**：面对一个新领域时，常常不知道从哪里入手。  

因此，我希望通过这个库：  
- 以统一的逻辑和符号体系 **重新梳理知识**；  
- 用记录来推动自我成长，**不断打磨理解**；  
- 给后来者提供一些 **坐标与参照**，让学习更顺畅。
