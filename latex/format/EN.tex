\section{Vector Space}
\subsection{Affine Sets and Convex Sets}
Given a real vector space $\mathbb{E}$, a set $S\subseteq \mathbb{E}$ is called an affine set if
$$\lambda \textbf{x} +(1-\lambda)\textbf{y}\in S$$
$\forall \textbf{x,y}\in S$ and $\lambda\in \mathbb{R}$\\
e.g. A hyperplane is a subset of $\mathbb{E}$ given by
$$H_{a,b}=\{x\in E:<a,x>=b\}$$
